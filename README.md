CineVault Pro: Elite Media Tracking
CineVault Pro is a high-performance, real-time media vault and competition ecosystem. Built for movie enthusiasts and digital archivists, it allows users to curate personal libraries, rate global cinema, and compete on a live leaderboard for the title of "Top Critic."

🚀 Core Features
Global Search & Discovery: Powered by the TMDB API, offering access to over 700,000+ films and series.

Live Leaderboard: A real-time competitive ranking system powered by Firebase Firestore's atomic increments.

The "Watched" Workflow: A frictionless logging system that prompts users for ratings before securing items to their private cloud vault.

Cloud Persistence: Fully synchronized data across devices via Firebase Authentication and Firestore.

Professional UX: Optimized with infinite scrolling, glassmorphism UI elements, and context-aware navigation.

🛠️ Technical Stack
Frontend: HTML5, CSS3 (Custom Grid & Flexbox), Vanilla JavaScript (ES6+)

Backend-as-a-Service: Firebase (Authentication, Firestore Database)

External API: The Movie Database (TMDB)
